UDPtime

UDPtime is a hobby project built to ping UDP protocols like Steam dedicated servers and display status using Uptime Kuma.

Steam

  • Endpoint: /steam/game-type-id/ip-adress
  • Args:
    • id: (requierd) game type id
    • ip: (requierd) server ip without port
  • Return: json
